# Skill: Kotlin Code Reviewer

## Review Kotlin code with focus on:


* Performance
* Coroutines best practices
* Android lifecycle safety
* Memory leaks
* Clean architecture violations

Check specifically:

* misuse of suspend functions
* blocking calls in main thread
* incorrect Flow usage
* ViewModel scope leaks
* unnecessary allocations

Output format:

* Critical issues
* Performance improvements
* Code readability suggestions
* Refactored code example
